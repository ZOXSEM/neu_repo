# Процедурная GTA на C++

## Введение

Этот проект представляет собой попытку создания процедурной версии игры в стиле Grand Theft Auto (GTA), полностью основанной на математических вычислениях и алгоритмах, без использования внешних ассетов (изображений, моделей и т.п.). Целью является демонстрация возможностей процедурного генерирования игровых миров с использованием C++.

## Архитектура проекта

Проект будет реализован как консольное приложение на C++ с использованием следующих основных компонентов:

- **Ядро игры** — управление циклом игры, обновлением состояния.
- **Генератор мира** — создание ландшафта, зданий, дорог и объектов.
- **Физика** — движение транспорта и персонажей.
- **ИИ** — поведение NPC.
- **Система ввода/вывода** — управление пользователем и отображение.
- **Рендеринг** — отображение мира в текстовом или графическом виде.

---

## Основные компоненты

### 1. Ядро игры

- Управление игровым циклом: обновление, рендеринг, ввод.
- Состояние игры: время, погода, игрок.
- Система событий и обработки действий пользователя.

### 2. Генератор мира

- Генерация ландшафта с помощью шума Перлина.
- Создание дорог, зданий и объектов.
- Использование фрактальных алгоритмов для генерации природных форм.
- Система тайлов (grid-based) для упрощения обработки.

### 3. Физика

- Простая система столкновений.
- Моделирование движения транспорта.
- Гравитация и движение по дорогам.

### 4. ИИ

- Поведение NPC: движение, взаимодействие с игроком.
- Система целей и задач.
- Простые алгоритмы поиска пути (например, A*).

### 5. Система ввода/вывода

- Обработка клавиш (WASD, стрелки).
- Отображение мира в консоли или через библиотеку (например, SFML).
- Визуализация игрока, транспорта и NPC.

### 6. Рендеринг

- Текстовый рендеринг (простой вариант).
- Графический рендеринг через SFML или OpenGL.
- Простые примитивы: прямоугольники, линии, текст.

## Генерация мира

### Ландшафт

- Используется шум Перлина для генерации высот.
- Плавные переходы между регионами.
- Разделение на зоны: город, деревня, дикая местность.

### Здания

- Генерация зданий на основе простых форм (прямоугольники).
- Случайное добавление окон, дверей, крыш.
- Использование трансформаций для повторяемости и вариативности.

### Дороги

- Генерация дорог с помощью алгоритмов построения путей.
- Создание перекрестков, поворотов.
- Случайное распределение дорожных знаков.

### Объекты

- Генерация деревьев, автомобилей, людей.
- Использование математических формул для распределения.

## Игровой процесс

### Цель

- Случайные миссии, выполнение которых дает награду.
- Система репутации: игрок может быть "преступником" или "полицейским".

### Миссии

- Простые задания: доставить предмет, убить цель, спастись от полиции.
- Система наград и штрафов.
- Динамическое изменение условий.

### Транспорт

- Генерация автомобилей.
- Управление транспортом игроком и NPC.
- Система торможения, ускорения, поворотов.

## Математические методы

### Шум Перлина

- Для генерации естественного ландшафта.
- Позволяет создать реалистичные изменения высоты.

### Фракталы

- Для генерации сложных форм (деревья, горы).
- Используются рекурсивные алгоритмы.

### Алгоритмы поиска пути

- A* или Dijkstra для навигации NPC.
- Учет препятствий и дорог.

### Матрицы и преобразования

- Для позиционирования и вращения объектов.
- Используются матрицы для 3D-преобразований (даже если вывод в 2D).

### Случайные числа

- Для генерации уникальных миров.
- Для случайного выбора задач, поведения NPC.

## Система управления

- Управление игроком: WASD + стрелки.
- Камера: следование за игроком или свободная.
- Интерфейс: отображение времени, денег, репутации.
