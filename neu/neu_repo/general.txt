Create a high-performance 2D top-down racing game engine using C++17 and OpenGL (GLFW/Glad) with zero external assets, where every visual element (cars, tracks, obstacles) is generated in real-time via mathematical SDF functions and procedural shaders. The engine must support a dynamic camera following the player, a procedural infinite track system, physics-based collisions, and a modular entity system for handling player-controlled vehicles, AI-driven enemies, collectible speed boosts, and hazardous obstacles.
